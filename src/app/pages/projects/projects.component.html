Talabat E-Commerce

.NET Core Web API — November 2024

Talabat E-Commerce is a scalable online shopping platform built with .NET Core Web API and SQL Server. The project features a robust architecture using Onion principles, ensuring maintainability and testability. Secure authentication is provided via JWT, with role-based access for users. Data operations are optimized through the Generic Repository and Unit of Work patterns, while Redis caching enhances performance for high-traffic endpoints. This solution delivers a seamless and secure experience for both customers and administrators.

.NET Core Entity Framework JWT Redis SQL Server

Company Management System

.NET Core MVC — October 2024

The Company Management System is a web application developed with .NET Core MVC, designed to streamline organizational processes. Leveraging N-Tier Architecture, the solution separates presentation, business logic, and data access for improved scalability. CRUD interfaces allow efficient management of employees and departments, while AutoMapper ensures smooth data transformation between layers. The system provides a user-friendly experience for administrators to oversee company operations with reliability and ease.

.NET Core MVC AutoMapper SQL Server

Restaurant (E-Commerce)

Angular — September 2024

Restaurant (E-Commerce) is a modern web application built with Angular, offering a responsive and engaging interface for food ordering. Tailwind CSS ensures seamless adaptability across devices, while RxJS powers efficient state management and real-time updates. During development, a local JSON server was used to simulate backend APIs, enabling rapid prototyping and integration. The project delivers a smooth user experience for customers browsing menus and placing orders online.
